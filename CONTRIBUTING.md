# Contributing

Thanks for helping improve Alice's Mirror.

## Before you start
- Check existing issues and discussions to avoid duplicates.
- For substantial changes, open an issue first so we can align on scope.

## Development
- Use Go 1.21+.
- Format code with `go fmt ./...`.
- Run tests with `go test ./...` if applicable (there are no automated tests yet).

## Pull requests
- Keep changes focused and well described.
- Include manual verification steps when tests are not available.
- Update documentation if behavior or flags change.
